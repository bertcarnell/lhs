## Installation

You can install the released version of lhs from [CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("lhs")
```

## Example

Create a random LHS with 10 samples and 3 variables 

``` r
X <- randomLHS(10, 3)
```
